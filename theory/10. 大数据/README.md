# 大数据发展
## Google三架马车
- GFS：分布式文件系统
- MapReduce：分布式计算，将大任务拆解成可以在多台机器上执行的小任务
- BigTable：Nosql
## Hadoop（雅虎开源）
- HDFS
- MapReduce
- YARN
## Hive
将SQL解析成MapReduce任务，运行在HDSF上。
## Spark
- RDD（弹性分布式数据集）
- 内存计算（速度快，但成本较磁盘高）
- DAG（MapReduce的Plus版本，根据RDD宽窄依赖划分Stage，多个Stage构成DAG，每个Stage为多个并行任务）
- SparkSQL（基于RDD和DAG）
- SparkStream（微批处理）
- SparkMLlib（丰富的机器学习库）
- SparkGraphX（图计算库）
